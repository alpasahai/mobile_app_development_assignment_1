## Milestone 1: Main Page Display
**Requirements**
- **Title:** At the top of the main page, prominently display the title "My Todo List."
- **Todo List Display:** The central part of the screen should feature a list of to-dos. For this milestone, you may hardcode a few to-do items to simulate how they will appear. This does not require dynamic loading or manipulation of data yet.
- **Add New Todo Button:** Include an "Add New Todo" button at the bottom of the screen. This button will be made functional in subsequent milestones, so it does not need to perform any actions at this stage.
